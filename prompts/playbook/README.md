# TBS Prompt Playbook — Phase 1

20 prompts adapted from the Expert-Driven Development toolkit for TBS-specific workflows. Designed for use on GenAI.mil (IL5, CUI-authorized) during Phase 1 of the Heywood Initiative.

## Usage
1. Open GenAI.mil and start a new conversation
2. Copy any prompt below and paste it into the chat
3. Replace `[BRACKETED_PLACEHOLDERS]` with your specific information
4. Review AI output critically — you are the expert, the AI is the tool

## Data Handling

- **CUI is authorized** on GenAI.mil (IL5)
- **PII must be anonymized** unless a PIA authorizes it — use duty positions and roles, not names
- **PHI is never authorized** on GenAI.mil
- **Classified data is never authorized** on any AI tool
- When in doubt, describe data types and field names — do not paste actual records
