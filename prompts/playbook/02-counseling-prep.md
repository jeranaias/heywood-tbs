# Prompt 02: Counseling Session Prep

**Use:** Prepare a structured counseling session for a student based on performance data
**Platform:** GenAI.mil
**Complexity:** Beginner
**Source:** Adapted from EDD Problem Definition Document (#2)

---

## Prompt

```
I am a Staff Platoon Commander (SPC) at The Basic School preparing a counseling session for a student.

EVALUATION POINT: [Week 12 Midpoint / Week 22 Final / Ad Hoc]
STUDENT CONTEXT (no names — use billet only):
- Current class standing: [TOP / MIDDLE / BOTTOM THIRD]
- Leadership grade trend: [IMPROVING / STABLE / DECLINING]
- Military skills grade trend: [IMPROVING / STABLE / DECLINING]
- Academics grade trend: [IMPROVING / STABLE / DECLINING]

KEY OBSERVATIONS:
- Strengths: [LIST_2_TO_3_OBSERVED_STRENGTHS — e.g., strong patrol orders, peers respect them, physical fitness above average]
- Areas for growth: [LIST_2_TO_3_OBSERVED_WEAKNESSES — e.g., hesitant under pressure, time management in planning, land nav accuracy]

SPECIFIC EVENTS:
[DESCRIBE_1_TO_2_RECENT_EVENTS_WHERE_PERFORMANCE_WAS_NOTABLE — good or bad]

Generate a counseling preparation document with:
1. **Opening statement** — Set the tone (developmental, not punitive). 2-3 sentences.
2. **Performance summary** — Factual overview of standing and trends across all three pillars.
3. **Strengths to reinforce** — What to acknowledge and encourage. Be specific to observations.
4. **Areas for development** — What to address. Frame as growth opportunities, not failures. Include specific behavioral examples.
5. **Development plan** — 2-3 concrete actions the student can take before the next evaluation point. Each should be measurable.
6. **SPC notes** — Talking points for the conversation (what to say, what to listen for, what questions to ask).

Keep the tone direct but professional. Marines respond to candor, not corporate language.
```

---

## Notes
- Use duty positions and class standing, NEVER student names
- This generates a prep document — the actual counseling session is a human conversation
- Review and personalize before the session; do not read AI output verbatim
